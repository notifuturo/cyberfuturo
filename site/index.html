Our commitments

  1. No forecasts. Every published claim is a measurement or a description of one.
  2. Numbers come from public APIs, never from an LLM. Editorial prose may be AI-assisted; numbers and charts are deterministic.
  3. Every number is traceable in three clicks — brief, canonical CSV, source methodology.
  4. Methodology is versioned. When a query changes, the version bumps and old values stay verifiable in git history.
  5. Sponsors never appear above the editorial. One sponsor per brief, never programmatic, never hidden.
  6. Subscriber data is never shared or sold — not to sponsors, not in aggregate, not for any reason.
