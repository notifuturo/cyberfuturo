HTTP: how the web talks.

Almost every modern app talks over HTTP. After this chapter, when someone says 'the API is returning 401,' you'll know exactly what happened and where to look.

What you'll understand

  • Why HTTP has verbs (GET, POST, PUT, DELETE) — and what each really means
  • What status codes tell you (200, 301, 404, 500) and how to read a header
  • The difference between a URL that serves HTML and one that serves JSON
