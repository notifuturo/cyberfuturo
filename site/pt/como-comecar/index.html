Como começar.

Três passos, zero instalação. Os exercícios rodam num Codespace do GitHub — uma máquina virtual no navegador, com Python, Git e SQLite prontos para uso.

Abrir no Codespaces Ver o repositório

Só precisa de uma conta gratuita no GitHub. 60h/mês de Codespaces grátis cobrem o curso inteiro.

Passo 1 — Abra o Codespace

Clique em Abrir no Codespaces. O GitHub cria uma máquina virtual com VS Code no navegador, o repositório já clonado e todas as ferramentas configuradas.

Prefere local? git clone https://github.com/notifuturo/cyberfuturo.git e abra a pasta no seu editor. Você vai precisar de Python 3.11+ instalado.

Passo 2 — Veja as lições

Com o terminal aberto no Codespace, rode:

bash — cyberfuturo/curriculum
user@cyberfuturo:~/curriculum$ ./cf list ▸ Lições disponíveis 00 Boas-vindas 01 Terminal 02 Primeiro commit em Git 03 Hello, Python ... Use: ./cf start NN para começar a lição NN

Passo 3 — Faça a primeira lição

Começa com a lição 00 e segue em ordem. Para cada lição:

  1. ./cf start NN

    Mostra a tarefa no terminal. Quando você quiser ver de novo, rode ./cf show NN.

  2. Faça o que a tarefa pede

    Cria arquivos, roda comandos, edita código. Tudo no mesmo terminal ou no editor do Codespace.

  3. ./cf check

    Valida o que você fez. Se passou, a lição fica marcada como concluída. Se faltou algo, o validador te diz o quê.

  4. ./cf next

    Pula pra próxima lição. Ou ./cf list pra ver todas de novo.

As tarefas ficam grátis pra sempre — o repositório é público. Os capítulos completos (as explicações, o porquê de cada comando, as analogias) fazem parte do curso: compre uma vez por $9 USD e tem acesso vitalício em 4 idiomas, mais handouts e certificados quando concluir cada módulo.
