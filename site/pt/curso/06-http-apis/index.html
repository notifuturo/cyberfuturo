HTTP: como a web conversa.

Quase toda aplicação moderna conversa por HTTP. Depois desse capítulo, quando alguém falar 'a API tá retornando 401', você vai saber exatamente o que aconteceu e onde olhar.

O que você vai entender

  • Por que HTTP tem verbos (GET, POST, PUT, DELETE) — e o que cada um significa de verdade
  • O que os status codes contam (200, 301, 404, 500) e como ler um header
  • A diferença entre uma URL que serve HTML e uma que serve JSON
