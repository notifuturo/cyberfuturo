  1. Você abre o projeto no GitHub Codespaces

    Um clique no botão acima. O GitHub te dá uma máquina virtual com VS Code, um terminal, Python, Git e SQLite já configurados. Nada pra instalar.

  2. Você executa ./cf start para sua primeira lição

    Um programinha (escrito em Python puro, sem dependências) mostra a tarefa, as instruções e as dicas. As lições vivem no terminal porque o terminal é o que você veio aprender.

  3. Você faz a tarefa com comandos reais

    Cada exercício é uma pequena missão: criar um arquivo, achar um escondido, escrever um script Python, rodar uma consulta SQL. Você usa as ferramentas reais, sem simulação.

  4. Você executa ./cf check para validar

    O validador olha o estado real do seu sistema de arquivos, do seu repositório Git, ou da saída do seu programa. Ele te diz o que deu certo e o que está faltando.

  5. Cada lição concluída é um commit Git

    Você aprende Git sem perceber: cada conquista fica registrada no seu histórico. Quando terminar o curso, você vai ter um repositório real com 30+ commits reais, pronto pra colocar no seu currículo.

Currículo — versão 0.1

Um subconjunto pequeno, ensinado com profundidade. Quanto mais longo o currículo, menor a chance de você terminar. A gente escolhe lições que são indispensáveis, não "nice to have".

  • 00
    Boas-vindas & orientação Seu Codespace, o terminal, pwd, ls, cat. Seu primeiro arquivo.
    ~15 min
  • 01
    Navegar o sistema de arquivos cd, ls -la, find, mkdir, mv, rm. Achar um arquivo escondido.
    ~25 min
  • 02
    Seu primeiro commit em Git init, status, add, commit, log, diff. Escrever uma mensagem de commit decente.
    ~30 min
  • 03
    Seu primeiro script em Python python3, variáveis, print, input, funções, rodar um script pelo terminal.
    ~30 min
  • 04
    Ramos e fusões em Git git branch, git switch, git merge, git log --graph. Primeiro merge de um ramo.
    ~30 min
  • 05
    Sua primeira consulta SQL sqlite3, CREATE TABLE, INSERT, SELECT, WHERE. Biblioteca de livros com filtros.
    ~35 min
  • 06
    HTTP e APIs públicas curl, JSON, um script Python que chama uma API real e extrai campos.
    ~30 min

Total: 7 lições · ~3h 15min de tarefas práticas.
