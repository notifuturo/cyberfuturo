About CyberFuturo.

Publish the canonical, free, rigorously documented set of leading indicators for how fast the digital, compute, and energy foundations of the near future are actually being built.

Everything CyberFuturo Index does is downstream of that one sentence. When a decision is ambiguous, the mission is the tiebreaker.

What that means in practice

CyberFuturo Index is a small, growing set of public, reproducible indices — each one a deterministic pipeline against a public data source, versioned methodology, no forecasts, no LLM-generated numbers. The first is arXiv AI research velocity. More follow the same pattern: public source, documented query, CSV and SVG outputs anyone can regenerate.

Findings are occasionally written up as dated briefs — short, numbered publications that cite specific index values and are never silently edited after the fact. See Issue #01.

Everything is governed by a small set of editorial rules: no forecasts, no LLM-generated numbers, sponsors (if any, eventually) never influence editorial content, and subscriber data is never shared.

Relationship to the CyberFuturo course

CyberFuturo also publishes a separate product — a $9 prerequisite course teaching terminal, Git, Python, and SQL to complete beginners. The course and the index publication share a brand and a domain, not an editorial process. The course sells a certificate; the index publication sells nothing today and answers to the rules above, not to a sales motion.

Who's behind it

CyberFuturo is an independent, solo-maintained project. Code and content are public: github.com/notifuturo/cyberfuturo.
