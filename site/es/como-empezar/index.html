Cómo empezar.

Tres pasos, cero instalación. Los ejercicios corren en un Codespace de GitHub — una máquina virtual en el navegador, con Python, Git y SQLite listos para usar.

Abrir en Codespaces Ver el repositorio

Solo necesitas una cuenta gratis de GitHub. 60 h/mes de Codespaces gratis cubren todo el curso.

Paso 1 — Abre el Codespace

Haz clic en Abrir en Codespaces. GitHub crea una máquina virtual con VS Code en el navegador, el repositorio ya clonado y todas las herramientas configuradas.

¿Prefieres local? git clone https://github.com/notifuturo/cyberfuturo.git y abre la carpeta en tu editor. Necesitarás Python 3.11+ instalado.

Paso 2 — Ve las lecciones

Con el terminal abierto en el Codespace, ejecuta:

bash — cyberfuturo/curriculum
user@cyberfuturo:~/curriculum$ ./cf list ▸ Lecciones disponibles 00 Bienvenida 01 Terminal 02 Primer commit en Git 03 Hello, Python ... Uso: ./cf start NN para iniciar la lección NN

Paso 3 — Haz la primera lección

Empieza por la lección 00 y continúa en orden. Para cada lección:

  1. ./cf start NN

    Muestra la tarea en el terminal. Si la quieres ver de nuevo, ejecuta ./cf show NN.

  2. Haz lo que la tarea pide

    Crea archivos, ejecuta comandos, edita código. Todo en el mismo terminal o en el editor del Codespace.

  3. ./cf check

    Valida lo que hiciste. Si pasó, la lección queda marcada como completada. Si falta algo, el validador te dice qué.

  4. ./cf next

    Salta a la siguiente lección. O ./cf list para verlas todas otra vez.
