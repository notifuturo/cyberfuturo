Python: hola, mundo.

Python no es un lenguaje misterioso. Es una herramienta que ejecuta instrucciones línea por línea. Si ya escribiste un comando en la terminal, ya sabes el 30% de Python.

Lo que vas a entender

  • La diferencia entre ejecutar Python en el REPL y ejecutar un archivo .py
  • Por qué print() tiene paréntesis en Python 3 (y no en Python 2)
  • Cuándo usar input() y cuándo no usarlo nunca
