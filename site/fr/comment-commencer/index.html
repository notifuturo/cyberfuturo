Comment commencer.

Trois étapes, aucune installation. Les exercices tournent dans un Codespace GitHub — une machine virtuelle dans ton navigateur avec Python, Git et SQLite déjà configurés.

Ouvrir dans Codespaces Voir le dépôt

Il te faut juste un compte GitHub gratuit. 60 h/mois de Codespaces gratuits couvrent tout le cours.

Étape 1 — Ouvre le Codespace

Clique sur Ouvrir dans Codespaces. GitHub crée une machine virtuelle avec VS Code dans le navigateur, le dépôt déjà cloné et tous les outils préconfigurés.

Tu préfères en local ? git clone https://github.com/notifuturo/cyberfuturo.git et ouvre le dossier dans ton éditeur. Il te faut Python 3.11+.

Étape 2 — Vois les leçons

Le terminal ouvert dans le Codespace, exécute :

bash — cyberfuturo/curriculum
user@cyberfuturo:~/curriculum$ ./cf list ▸ Leçons disponibles 00 Bienvenue 01 Terminal 02 Premier commit Git 03 Hello, Python ... Usage : ./cf start NN pour commencer la leçon NN

Étape 3 — Complète la première leçon

Commence par la leçon 00 puis enchaîne dans l'ordre. Pour chaque leçon :

  1. ./cf start NN

    Affiche la tâche dans le terminal. Pour la revoir, exécute ./cf show NN.

  2. Fais ce que la tâche demande

    Crée des fichiers, exécute des commandes, édite du code — tout dans le même terminal ou l'éditeur du Codespace.

  3. ./cf check

    Valide ton travail. Si tout passe, la leçon est marquée complétée. Sinon, le validateur te dit ce qui manque.

  4. ./cf next

    Passe à la leçon suivante. Ou ./cf list pour toutes les revoir.
