Confidentialité

CyberFuturo collecte le minimum possible. La progression du cours est opt-in et anonyme ; les données d'achat se limitent à ce qui est nécessaire pour vous donner accès — et elles sont effaçables.

Ce que nous collectons (seulement si vous acceptez)

  • Quand vous commencez une leçon (slug de la leçon, ex. 01-terminal)
  • Quand vous terminez une leçon (./cf check réussit)
  • Un identifiant anonyme aléatoire, généré une fois dans votre environnement, stocké dans curriculum/.progress.json
  • La langue de la leçon que vous lisez (pt, es, en, fr)
  • Le pays que Cloudflare détecte en périphérie (BR, MX, FR, ...) — l'IP elle-même est rejetée

Ce que la télémétrie de progression ne collecte JAMAIS

  • Votre compte GitHub
  • Votre code, vos fichiers, les messages d'erreur
  • Votre IP (Cloudflare lit le pays en périphérie et rejette l'IP avant qu'elle n'arrive dans notre stockage)
  • Tout ce qui, dans la télémétrie, pourrait vous identifier individuellement

Votre nom et votre e-mail ne font pas partie de la télémétrie — nous ne les recevons que lorsque vous achetez le cours (voir ci-dessous). Si vous exécutez ensuite ./cf activate pour récupérer votre certificat, votre progression jusque-là anonyme est associée à cet achat afin que nous puissions l'émettre.

Ce qui arrive aux données

  • Stockées dans une base de données Cloudflare (D1) contrôlée par CyberFuturo
  • Utilisées pour générer le tableau de bord public /stats/ (uniquement des totaux agrégés, jamais par personne)
  • Jamais vendues, jamais partagées avec des sponsors, jamais croisées avec des tiers

Quand vous achetez le cours

Un achat est différent de la progression anonyme ci-dessus. Pour vous donner accès et vous envoyer votre identifiant, nous collectons :

  • Votre nom et votre e-mail, tels que saisis au paiement Stripe
  • Les métadonnées de paiement de Stripe : montant, devise et l'id de la session de paiement — nous ne voyons ni ne stockons jamais votre numéro de carte
  • Votre préférence de langue et les identifiants d'accès que nous générons pour vous

Base légale : nous traitons ces données pour exécuter notre contrat avec vous — vous livrer le cours que vous avez payé (RGPD Art. 6(1)(b) / LGPD Art. 7(V)).

Durée de conservation : tant que vous avez accès au cours. L'enregistrement de la transaction est conservé jusqu'à 5 ans pour répondre aux obligations fiscales et comptables, puis supprimé. Vous pouvez demander la suppression de vos données plus tôt — sauf l'enregistrement de la transaction que nous sommes légalement tenus de conserver.

Avec qui nous partageons les données

Nous ne vendons jamais vos données. Un petit ensemble de prestataires de confiance les traitent pour notre compte, chacun selon ses propres conditions de traitement des données :

  • Stripe — traite votre paiement. Politique de confidentialité
  • Resend — livre votre e-mail de bienvenue/d'accès. Politique de confidentialité
  • Cloudflare — héberge le site et la base de données et détecte votre pays en périphérie. Politique de confidentialité
  • GitHub — fournit l'environnement de pratique Codespaces, uniquement si vous l'utilisez. Politique de confidentialité

Certains de ces prestataires opèrent hors de votre pays ; lorsqu'un transfert international de données personnelles a lieu, il s'appuie sur les garanties offertes par chaque prestataire.

Vos droits — désactiver ou supprimer

Progression du cours (télémétrie) :

  • ./cf telemetry off — désactive les pings à partir de maintenant
  • ./cf telemetry forget — supprime votre anon_id local et efface les enregistrements du serveur
  • Ou ouvrez une issue sur github.com/notifuturo/cyberfuturo demandant la suppression de votre anon_id

Données d'achat (nom, e-mail) : écrivez à ola@cyberfuturo.com (ou répondez à votre e-mail de bienvenue) et nous supprimerons vos données personnelles sous 30 jours, en ne conservant que ce que la loi fiscale exige. Vous pouvez aussi demander une copie de vos données et déposer une réclamation auprès de votre autorité de protection des données (en France, la CNIL).
