Confidentialité

CyberFuturo collecte le minimum possible. Tout est opt-in, anonyme et effaçable.

Ce que nous collectons (seulement si vous acceptez)

  • Quand vous commencez une leçon (slug de la leçon, ex. 01-terminal)
  • Quand vous terminez une leçon (./cf check réussit)
  • Un identifiant anonyme aléatoire, généré une fois dans votre environnement, stocké dans curriculum/.progress.json
  • La langue de la leçon que vous lisez (pt, es, en, fr)
  • Le pays que Cloudflare détecte en périphérie (BR, MX, FR, ...) — l'IP elle-même est rejetée

Ce que nous ne collectons JAMAIS

  • Votre e-mail, nom, compte GitHub
  • Votre code, vos fichiers, les messages d'erreur
  • Votre IP (Cloudflare lit le pays en périphérie et rejette l'IP avant qu'elle n'arrive dans notre stockage)
  • Tout ce qui pourrait vous identifier individuellement

Ce qui arrive aux données

  • Stockées dans une base de données Cloudflare (D1) contrôlée par CyberFuturo
  • Utilisées pour générer le tableau de bord public /stats/ (uniquement des totaux agrégés, jamais par personne)
  • Jamais vendues, jamais partagées avec des sponsors, jamais croisées avec des tiers

Comment désactiver ou supprimer

  • ./cf telemetry off — désactive les pings à partir de maintenant
  • ./cf telemetry forget — supprime votre anon_id local et efface les enregistrements du serveur
  • Ouvrir une issue sur github.com/notifuturo/cyberfuturo demandant la suppression de votre anon_id fonctionne aussi
