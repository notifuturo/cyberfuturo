CHAPITRE 02 · 18 min · CHAPITRE PAYANT · EXTRAIT

Ton premier commit Git.

Tout vrai travail finit dans Git. Ce chapitre te fait passer de « j'ai copié le fichier dans un autre dossier pour sauvegarder une version » au bon modèle mental : Git est une ligne du temps et chaque commit est un point dessus.

Ce que tu vas comprendre

  • Pourquoi git init crée un dossier invisible appelé .git
  • Ce que git add fait que git commit ne fait pas (et inversement)
  • Comment écrire un message de commit que ton toi-du-futur ne va pas haïr
🔒
FIN DE L'EXTRAIT
Ce chapitre continue avec :
  • l'anatomie d'un commit sous le capot (hash, arbre, auteur, timestamp)

  • les trois façons de défaire (et celle que tu vas utiliser 95 % du temps)
